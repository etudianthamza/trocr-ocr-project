#  TrOCR Project

## 📌 Description
This project implements an Optical Character Recognition (OCR) system based on TrOCR, a Transformer-based model designed to extract and recognize text from images. It includes preprocessing, model inference, and experimentation workflows.

## 🚀 Technologies
- Python
- PyTorch
- Hugging Face Transformers

## ▶️ Usage
Open the notebook:

Tr_OCR_Algorithm.ipynb

Run all cells to test the OCR pipeline.

## 📊 Results
The notebook includes examples of input images and extracted text.
